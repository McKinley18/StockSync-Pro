# StockSync Pro

StockSync Pro is a professional-grade pantry management application built with React Native and Expo. It helps users track inventory, manage meal plans, and reduce food waste through smart analytics and shopping list integration.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or newer)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
To start the web preview:
```bash
npx expo start --web
```

To run on iOS or Android (requires physical device or simulator):
```bash
npx expo start --ios
# or
npx expo start --android
```

## ✨ Features
- **Smart Dashboard:** Real-time stats on inventory value, waste, and upcoming expirations.
- **Inventory Management:** Categorized list with low-stock alerts and smart autocomplete.
- **Weekly Meal Planner:** Sync recipes with your pantry to automatically generate shopping lists.
- **Financial Insights:** Detailed reports on food waste and purchase habits.
- **Recipe Book:** Save your favorite recipes and track required ingredients.

## 🎨 Design System
- **Theme:** Professional Slate & Blue palette.
- **Responsive:** Permanent sidebar for desktop/tablets and a front-drawer for mobile.
- **Localized:** Strictly uses MM-DD-YYYY date formatting for all user inputs.

## 🛠 Tech Stack
- **React Native & Expo SDK 55**
- **Expo SQLite** for native storage
- **Lucide Icons** for a modern interface
- **React Navigation** for robust routing
